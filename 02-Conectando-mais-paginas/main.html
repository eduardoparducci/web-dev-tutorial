Conectando Mais Páginas

O objetivo desse exercício é mostrar como são feitas as conexões entre diferentes páginas de um site, Além disso, vamos mostrar como são feitos os links para páginas de websites completamente diferentes.


Link para uma página de um website externo (Google, YouTube, etc...)

Em um website, normalmente encontramos botões que nos direcionam para outras páginas. Iremos entender como essa navegação é feita.

Como essa é uma ação extremamente comum nos sites, o HTML possui uma tag específica par iasso. a tag 'a'.

Podemos, por exemplo, criar um link que nos leva para a página inicial deste repositório. Veja o exemplo abaixo:

Visite o repositório deste projeto!

Note que dentro da tag 'a', temos dois parâmetros. href e target. Cada um deles tem uma função.

href: Contém o URL (endereço) da a página externa.

target: Descreve a maneira como essa página será aberta. Neste exemplo, definimos a página para ser aberta em uma nova guia do seu navegador. Tente remover completamente o parâmetro target do link acima e veja o que acontece.


Link para uma página dentro do mesmo website

Muitas vezes, navegamos para páginas que estão dentro do mesmo website. Iremos agora navegar para uma segunda página. Dentro desta mesma pasta, existe um arquivo chamado 'secondary.html'. Iremos criar um link de navegação para ela Aqui

O href do último link possui um '.' que serve para referenciar a localização do arquivo que estamos tentando abrir. O ponto representa a pasta atual, neste caso a pasta '02-Conectando-mais-páginas'. Conforme o site toma corpo e vai crescendo, surge a necessidade de agruparmos algumas páginas em pastas diferentes. Para referenciar outras pastas, podemos utilizar '..' que indica que estamos retornando a pasta anterior, neste caso a pasta 'web-dev-tutorial'.
