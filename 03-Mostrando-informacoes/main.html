Mostrando Informações

O objetivo desse exercício é apresentar algumas tags que auxiliam no processo de exibição e apresentação de informações em um website. Iremos apresentar aqui, como podemos montar:

  • Listas não ordenadas
  • Listas ordenadas

Listas não ordenadas

É exatamente a lista que foi mostrada acima, onde a ordem dos itens não importa. Podemos modificar um pouco a aparência dessas listas através do HTML. Veja:

  • Lista
  • com
  • indicadores quadrados

Listas ordenadas

O nome é autoexplicativo. Quando a ordem importa. Podemos modificar uma lista não ordenada para uma lista ordenada trocando a tag 'ul' para 'ol', onde ol significa ordered list.

  1. Lista
  2. ordenada

Exercícios

  1. Todo desenvolvedor, em algum momento, irá se deparar com uma situação que nunca trabalhou antes. Sendo assim, saber pesquisar e interpretar os resultados é essencial. Realize uma pesquisa e exiba na tela uma lista ordenada que contém todos os meses do ano, com indicadores em número romando (I. Janeiro, II. Fevereiro, ..., XII. Dezembro)

  2. Tente mesclar o conteúdo deste tópico (Mostrando informações) com o tópico passado (Conectando mais páginas) e crie uma lista não ordenada que contém links para todas as lições feitas até o momento (Primeiro website, ...). Dica: para criar um link dentro de um item de uma lista, pense que dentro da tag 'li' podemos ter uma tag 'a'

  3. Crie uma cópia da receita de Pão de queijo 3 ingredientes
    Dica: Você pode usar as tags de título (h1, h2, h3, ... ) para indicar as seções da receita (ingredientes, modo de preparo), listas não ordenadas e ordenadas para cada item de cada seção.

  4. Como temos uma lista de perguntas (que está estruturada como uma lista ordenada), estruture a lista de respostas como uma lista ordenada também.

Bônus: Para revisar o exercício passado, depois de responder às perguntas, ao final da página (após o texto 'Menu de navegação', coloque um link com o texto 'Anterior' que nos leva até a página do tópico 02 e adicione no final da página do tópico 2, um link com o texto 'Próximo' que nos traz de volta para essa página.
