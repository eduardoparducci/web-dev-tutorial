O objetivo desse exercício é te familiarizar com as principais tags de texto do HTML.

Uma tag é definida por um texto envolto pelos caracteres '<' e '>'

Algumas tags exigem fechamento, outras não. O fechamento de uma tag é envolto pelos caracteres '<' e '/>'. a tag 'br' significa 'line break' e pula uma linha na página, observe que essa tag não exige fechamento.

Por exemplo. A tag 'h1' é utilizada para representar títulos principais da página. Essa tag exige fechamento. O título da página que você está lendo foi escrito utilizando a tag 'h1'

Além do título principal 'h1' temos títulos secundários representados por 'h2', 'h3', ... , 'h7'. Tente modificar o título principal dessa página para 'h2' e observe o resultado.
